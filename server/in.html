Abstract

What is FinWise?

FinWise is a hierarchical multi‚Äëagent platform that provides personalized, context‚Äëaware financial advice by combining specialized AI agents under a central orchestration layer. The system ingests transaction and profile data, decomposes advisory tasks (budgeting, debt optimization, investment suggestion, financial education), and synthesizes agent outputs into actionable plans tailored to user goals and risk tolerance.

We present the architecture, agent designs, orchestration strategy, privacy and safety mechanisms, and a mixed automated + human evaluation methodology. Experiments on synthetic and anonymized real transaction datasets demonstrate improvements in recommendation coherence, plan feasibility, and user satisfaction compared to single‚Äëmodel baselines. We discuss limitations, deployment considerations, and directions for future work.
